# 635法

> 以他人的意見為脈絡，接續著發想
> 六個人、一次三個想法、每次五分鐘
![](https://i.imgur.com/30RktGM.jpg)

### 使用方法
  1. 訂定題目並確定所有人能理解問題
  2. 以五分鐘為單位傳寫不同人的表單
  3. 30分鐘結束後，共可得到108個創意設想
  4. 進行歸納討論

### 使用時機
  - 已有目標方向，為了製造有趣、有創意的解決方法時

### 使用工具
  1. 筆
  2. 計時器
  3. 635表單
![](https://i.imgur.com/bxkZg2t.jpg)




### 優缺點
- 優點
    - 思考脈絡會因為成員特性、以及成員間的碰撞，而產生出團隊特色
    - 在短時間大量產出創意
    
- 缺點
    - 若有人提出的想法特別不適用，討論可能將受到影響

```
    可以依循其他人脈絡延伸想法，不容易偏離主軸

```
