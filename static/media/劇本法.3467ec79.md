# 劇本法

> 從訪問過程中收集關鍵活動故事，洞察使用者行為背後的意義




### 使用方法
1.從訪談紀錄中挑出關鍵活動
2.將人、事、情境寫成劇本表單
3.進行分析(ex.正面?負面?)

### 使用時機
訪談後，幫助重現田野研究中使用者經驗的活動細節
### 準備工具
1.電腦
2.紙
3.筆
4.訪談大綱
5.劇本表格
![](https://i.imgur.com/FTjkyga.jpg)


### 優缺點
- 優點
    - 幫助研究員抓住細節
    - 透過情境故事分析背後意義

- 缺點
    - 要懂得觀察找到細節否則難以進行
    - 有些情境難以分析需求與機會
